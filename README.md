# Apache-Guacamole-1.6.0-Ubuntu-24.04-Install-Script
Ubuntu 20.04 Install Script for Apache Guacamole 1.6.0, Tomcat 10, MariaDB 11.8, MySQL Connector 9.6.0 and Apache Jakarta EE Migration Tool 1.0.10

## Usage

You have to run:

```
curl -sSL https://raw.githubusercontent.com/Michdo93/Apache-Guacamole-1.6.0-Ubuntu-24.04-Install-Script/refs/heads/main/install_guacamole.sh | sudo bash
```
